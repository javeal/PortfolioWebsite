Facial Recognition System

Description:

This facial recognition system was implimented using the PCA algorithim. The PCA algorithim is a machine learning technique that makes use of EigenFaces, Euclidean Distance and data compression. This system was developed in Python using the Spyder IDE from Anaconda. The system is only currently functional in the Spyder IDE. The next step is to use python to run the system in Windows PowerShell. This will allow the system to run with a GUI and access the webcam of a computer.

Skills:

Machine Learning, Python, Digital Signal Processing, Linear Algebra

Responsive Vehicle

Description:

The objective was to navigate through an obstacle course, without being able to look at the obstacle course. The project was done using a Rumba Vaccum fitted with a Texas Instruments Micro-Controller and a variety of sensors. This project was coded in C and made use of GUI to navagate the robot. The robot collected data from the sensors and sent the data to the GUI through an SSH connection. The robot was configured to connect to wifi to make the SSH connection possible. Commands were sent to the robot through the connection once data had been recieved by the GUI. Some of the obstacles included cliffs, tall poles, short poles and an out of bounds line.

Skills:

Embedded System Design, C Programing, Bitwise Register Operations and Teamwork

DTMF Encoder/Decoder

Description:

DTMF is the logic of encoding frequencies for dial up calls. The various dial tones are a combination of of two frequencies, and those frequencies depend on the location of the number on the dial pad. This project required the design of very narrow band pass filter that could decode the frequencies the system recieved. The signals were incoded by the generation of sinusoids based on the number entered. The system produced the tones and delivered the decoded results. This was done using MATLAB.

Skills:

Filter Design and MATLAB

DTMF encoder/decoder Report

Random Number Generator

Description:

The Random Number Generator is a project developed using Altera products in the area of digital circuit design. The numbers were generated using a formula that produced a output that varied depending on the input. The design process included creating state machines, truth tables, and circuit diagrams in altera. For convience, some of the components were coded using Verilog. Some components inside the Generator were Register Files, Registers and a parallel shift Register. The input was controlled with switches on an Altera board, and the output was shown on 7 segment displays.

Skills:

Digital Circuit Design, Boolean Logic and Verilog

Sound System Reciever

Description:

This project can be consisted of the design of 3 analog filters. There was a second order active low pass filter, a second order active high pass filter, and a first order active band pass filter. The idea of the filters was to divide parts of the signal to send to different types of speakers. For example, low frequencies would be directed to a subwoofer. Some of the components used to create this system were op amps, capacitors, inductors and resistors.

Skills:

Analog Circuit Design, Circuit Analysis, Filter Design and Testing
